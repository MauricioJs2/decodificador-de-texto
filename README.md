# ProjetoDecodificador

Funções:

- Codificar o texto inserido, somente em letras minúsculas

- Mostrar texto decodificado

- Possibilitar copiar o texto decodificado

Linguagens utiluzadas:

- JavaScritp
- HTLM
- CSS
